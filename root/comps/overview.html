Why Go4Lage?

With Go4lage, you can take web applications from concept to launch in just a matter of days. It takes care about the basic hassle and let's you focus on your app, without introducting too much complexity. On top the result will be actually fast and easily deployed.

Features of Go4lage

Build in cached user management system with an admin dashboard.

Designed to ship in an instant with docker compose.

Backup system in place. Designed for maximum air-gap security

Reduced complexity with no attempt to hide it's magic.

Two Factor System optionally implemented for admins.

Planned features include a proper, well designed test suite as well as SEO SSR. Feel free to contribute!

Example cost calculation

Server costs are an essential part of any business. But how much is it? This is an example calculation.

You need a main server. And a test system. The test system will mirror the latest updated. Do not be like crowdstrike or Dell, use a test system that's similar to the production. Github is free for up to 5 persons as your vcs. Many services need also email. Unfortunately I have to recommend using a payed service as sending emails reliably is hard.

  • Main server 10 to 25 Euro a month with domain.
  • Test server 10 Euro a month with domain.
  • Email service 10 to 15 Euro a month.

that's aproximately 30 to 50 Euro a month. Every company should be able to afford this.

Furthermore it is sometimes not about scaling up, if you are in need for several sepparate instances for different customers, you can do this with docker compose. So your costs will not even go up until you hit a large amount of total users.

FAQ

Question Explanation
What does Go4lage mean? Go4lage is a play on words. 'Vorlage' (4lage) is German for template, so it could roughly be interpreted as 'GoTemplate.'
Will there ever be Redis support? Redis support is not planned as Go offers a more native caching solution. Alternatively, you can use Postgres in memory mode for SQL features.
Is Go4lage really free? Yes, Go4lage is free both as in free speech and as in free beer, even for commercial use.
How can I contribute? Please refer to the official GitHub repository for details.
Why is 2FA available for superusers but not for normal users? 2FA requires a well-thought-out setup. It involves more than just providing a key secret to users. To secure your app effectively, proper planning is necessary. If you consider this, you will be able to configure Go4lage accordingly.
I come from Django (Python), is Go hard to learn? Not at all. Only the initial steps might seem challenging. Golang values simplicity and is often more straightforward than Python because it avoids unexpected complexities.
Why is there no ORM? ORMs introduce an unnecessary layer of complexity. We believe in starting with the data you want to manipulate, as the most important aspect of information technology is the data stored in the database.
Why is the JS build included and tracked? This build was included to allow users to avoid building it themselves. Many might not need to touch it at all.
How powerful (fast) is Go4lage? Although there are no real-world tests published yet, Go applications are generally fast enough. The main appeal of Go is not just its speed but its simplicity and the ability to write concurrent code, which aids in rapid development and producing robust code that can adapt to future requirements.
